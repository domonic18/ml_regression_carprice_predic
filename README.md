## 项目背景
在当今的汽车市场中，二手车的价格受到多种因素的影响。通过机器学习技术，我们可以分析历史数据，建立模型，从而更准确地预测二手车的价格。这不仅可以帮助消费者做出更明智的购买决策，也可以为卖家提供合理的定价参考。

## 使用方法
1. 安装依赖
```
pip install -r requirements.txt
```
2. 运行项目
```
streamlit run app.py
```
